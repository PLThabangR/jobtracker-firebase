# A simple job tracking application built with **React** and **TypeScript**, powered by a mock backend using **JSON Server**.

---
## step 1
- Clone this project using git clone 
- git clone <repository_url>
- cd job-tracker

## Step 2 
- npm install  
## Step 3 
- To run app
- npm start
